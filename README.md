<h3 align="center">Tuner-ReactNative</h3>


- 使用ReactNative实现的调音器软件
- 支持ios和android
  
#### 项目启动
  - ```git clone https://github.com/Tourscholar/Tuner-ReactNative.git ```		克隆本仓库到本地
  - ```yarn ios```    编译启动ios版本
  - ```yarn android```    编译启动android版本
